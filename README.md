A Platform for Adopting Pets with Disabilities

This is a prototype web application I'm developing for a team project. It demonstrates how technology can help connect special needs pets with loving families.

Project Overview
PetMatch is a Single Page Application (SPA) prototype that facilitates the adoption of pets with disabilities. The platform serves as a bridge between potential adopters and animal shelters/NGOs, focusing specifically on pets that often get overlooked due to their special needs.

Current Status: This is a functional prototype demonstrating core features and user flows.

Core Purpose
The application aims to:

Make pets with disabilities more visible to potential adopters
Simplify the adoption process through technology
Provide comprehensive information about each pet's specific needs
Connect users directly with verified animal shelters

Key Features Implemented
Pet Discovery
Browse pets with various disabilities (blind, deaf, tripod, etc.)
Filter by location (CEP/city), pet type, size, and age
View detailed profiles with photos and descriptions

User Experience
Save favorite pets (stored locally in browser)
Recently viewed pets history
Dark/Light mode toggle
Responsive design for all devices

NGO/Shelter Integration
Interactive map showing partner locations
Direct contact forms for shelters
Shelter ratings and distance information

Design System
Custom color palette focused on accessibility
Modern, clean interface with clear visual hierarchy
Smooth animations and transitions
WCAG-compliant accessibility features

Technical Implementation
Architecture
Pure JavaScript SPA (no frameworks)

Client-side routing
Component-based architecture
Local storage for user preferences

Tech Stack
HTML5 - Semantic markup
CSS3 - Custom properties, Flexbox, Grid
Vanilla JavaScript - ES6+ features, Classes
Leaflet.js - Interactive maps
Font Awesome - Icons
